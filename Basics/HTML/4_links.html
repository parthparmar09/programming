Links in HTML

There are many ways to create HMTL links of many types. The most used ones are :

  • Simple text link: Click me

  • Can wrap anything inside the a tag and make it a link, for ex: images

  • Relative links - to open pages within the same website A HTML page

  • Email links - opens default email client with a pre-filled email address Mail me

  • Telephone links - initiate a phone call when clicked on mobile devices Call me

  • Empty links - doesn't do anything Click here

  • Link to a specific section in a page by its id Go to top

  • Link to download a file Download image

There are many ways to open HMTL links in many ways using target attribute :

  • Opens in the same frame or window(default): _self

  • Opens in a new, unnamed browser window or tab: _blank

  • Opens in the parent frame or window of the current frame or window: _parent

  • Opens in the top-level frame or window, replacing all other frames or windows, if applicable: _top
