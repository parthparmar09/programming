# ECMAScript (ES) Timeline & Major Features

## 1. What is ECMA, ES, and JavaScript
- **ECMA**: Organization that standardizes programming languages and specs.
- **ECMAScript (ES)**: The official standard of JavaScript.
- **JavaScript**: Programming language implementing ECMAScript. New JS features come from ES proposals ratified by ECMA.

## 2. Major ECMAScript Releases

| Year  | Version      | Key Features / Updates |
|-------|-------------|----------------------|
| 1997  | ES1         | First official standard. Basic JS. |
| 1998  | ES2         | Minor editorial changes. |
| 1999  | ES3         | Regular expressions, `try/catch`, better string handling. |
| 2000  | ES4         | **Abandoned** due to complexity. |
| 2009  | ES5         | `"use strict"`, `JSON` support, `Array.prototype.map/filter/reduce`, getters/setters. |
| 2015  | ES6 / ES2015 | `let`, `const`, arrow functions, classes, modules (`import/export`), template literals, promises, default/rest/spread parameters, `Map`, `Set`. |
| 2016  | ES7 / ES2016 | `Array.prototype.includes`, exponentiation operator `**`. |
| 2017  | ES8 / ES2017 | `async/await`, `Object.values()`, `Object.entries()`, string padding, shared memory & Atomics. |
| 2018  | ES9 / ES2018 | Rest/spread for objects, asynchronous iteration (`for await...of`), `Promise.finally`. |
| 2019  | ES10 / ES2019 | `Array.flat()`, `Array.flatMap()`, `Object.fromEntries()`, `String.trimStart/trimEnd()`. |
| 2020  | ES11 / ES2020 | `BigInt`, dynamic `import()`, nullish coalescing `??`, optional chaining `?.`, `globalThis`. |
| 2021  | ES12 / ES2021 | Logical assignment operators (`&&=`, `||=`, `??=`), numeric separators `_`, `Promise.any`, `WeakRefs`. |
| 2022  | ES13 / ES2022 | Top-level `await` in modules, class fields, `Error` cause, RegExp match indices. |
| 2023  | ES14 / ES2023 | `Array.findLast()` / `findLastIndex()`, Hashbang (`#!`), change array by copy, `Symbol.prototype.description` updates. |


### Notes
- **ES6 / ES2015** is the biggest modern update; all features after are mostly incremental.
- Modern JS development mostly uses **ES6+** features.
- New ECMAScript features are ratified yearly and gradually implemented in browsers and Node.js.
